# Placeholder Content Files

This directory contains placeholder markdown files for all topics in the D1 Leadership AI Deep Dive presentation.

## Purpose

These files serve as:
- Initial content when topics haven't been filled with real content yet
- Reference templates showing the expected topic ID and title
- Downloadable files for admin users to use as starting points

## Usage

As an admin:
1. Navigate to any topic in the presentation
2. Click the green download button (⬇) to download the current content
3. Edit the downloaded markdown file with your actual content
4. Click the purple edit button (✎) to upload your updated content

## File Naming

Each file is named `{topicId}.md` matching the topic ID defined in [ui/app/data/topics.ts](../../ui/app/data/topics.ts).
